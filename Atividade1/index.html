Tag <body> é obrigatória?

A tag body do HTML é ogrigatória, porém se se não colocar, o código não vai dar erro, pois o próprio navegador adiciona quando não tem. Apesar disso, é boa prática sempre utilizar a tag <body>.

Tag <pre> do HTML

A tag <pre> é usada para exibir texto pré-formatado, ou seja, ela mantém exatamente o que você escreveu no código, incluindo espaços, quebras de linha e indentação.

Exemplo:

def imprimir_padrao(num_linhas):

                        for num_colunas in range(num_linhas-i):
